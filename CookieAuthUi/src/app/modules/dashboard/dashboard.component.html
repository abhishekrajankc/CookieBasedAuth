GraphQL API Overview

GraphQL lets clients request exactly the fields they need from a single endpoint. This reduces over-fetching and keeps frontend payloads small and focused.

Query

Read data with precise field selection.

Mutation

Create/update/delete data from one endpoint.

Subscription

Real-time updates over a persistent connection.
